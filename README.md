NAME
====

EERPG - Emergent Economies for Role Playing Games

SYNOPSIS
========

    use EERPG;

DESCRIPTION
===========

EERPG is an implementation of the "Emergent Economies for Role Playing Games" white paper by Jonathan Doran and Ian Parberry.
